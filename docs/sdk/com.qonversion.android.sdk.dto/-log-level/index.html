This enum contains available settings for LogLevel. Provide it to the configuration object via Qonversion.initialize while initializing the SDK or via Qonversion.setLogLevel after initializing the SDK. The default value SDK uses is LogLevel.Info.

You could change the log level to make logs from the Qonversion SDK more detailed or strict.
